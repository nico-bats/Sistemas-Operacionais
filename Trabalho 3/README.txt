Descrição:

	Este trabalho trata-se de um simulador de gerenciamento de memória através do algoritmo Relógio
implementado em C juntamente com a lista encadeada dinamica circular.Os parametros de compilação estao no arquivo
Makefile, bastando apenas utilizar a diretiva "make all" para compilar e "make run" para executar.

O algoritmo irá procurar por um arquivo de entrada chamado "input.exe" o qual esta junto com o programa e deve ser mantido o seu
padrao interno, caso contrário a leitura dos dados dos processos será incorreta.
